# Mirai-Automatic-Slider-Captcha-Solver
Mirai 全自动滑块验证

### 简介

此插件通过服务器进行滑块验证，用户无需执行任何操作即可通过滑块验证。

请注意，此插件仍处于测试阶段，不推荐大规模生产中使用。

### 使用方法

请下载Github Releases中的jar文件，并将其加入运行时classpath或放入plugins文件夹

### 警告

使用此插件会将登陆账号的滑块验证URL以加密方式发送到溯洄的服务器上进行验证！
